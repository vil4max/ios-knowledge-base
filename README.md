# iOS Interview Prep

Personal knowledge base for Senior iOS interviews — Swift, iOS SDK, architecture, system design, networking, AI.

**Browse:** [vil4max.github.io/ios-knowledge-base](https://vil4max.github.io/ios-knowledge-base/)

Use sidebar or search to find a topic and refresh before an interview.

## GitHub Pages setup

1. Repo **Settings → Pages**
2. **Build and deployment → Source:** Deploy from a branch
3. Branch: `main`, folder: `/ (root)`
4. After push, site at `https://vil4max.github.io/ios-knowledge-base/`

Docsify (`index.html` + `.nojekyll`) — no build step.

## Local preview

```bash
npx serve .
# open http://localhost:3000/ios-knowledge-base/  (or use any static server)
```

## Add a topic

1. `section/topic/README.md`
2. Update `_sidebar.md`
3. `git push`
